# ReklamnoeAgentstvoAPI

Для работы API необходимо запустить отладку в Visual Studio и открыть любой интересующий html-файл.

API работает с данной базой и всеми новыми формулировками, написанными в БД.

Чтобы зайти в админ-панель, необходимо пройти авторизацию: Данные для вход храняться в базе данных "Учётная_запись".

Администратор имеет возможность удалять записи в БД.

Весь управленческий персонал имеет свой функионал и права доступа при работе с админ-панелью.
